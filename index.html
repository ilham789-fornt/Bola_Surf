3D Bola Lari

BOLA JALAN

Lewati rintangan, kumpulkan koin, dan capai garis finish di lapangan bola.

Permainan pengaturan

Level 2: Tantangan seimbang. Kecepatan standar.

Tutorial

  • ArrowLeft / ArrowRight atau A / D untuk pindah jalur.
  • ArrowUp, W, atau Space untuk melompat.
  • ArrowDown atau S untuk turun cepat saat sedang melompat.
  • Di HP: tap bagian atas untuk lompat, bawah kiri/kanan untuk pindah jalur.

Rules

  • Hindari cone, barrier, dan spike. Tabrakan membuat game over.
  • Untuk papan gantung merah-putih, tetap di tanah. Melompat saat melewatinya membuat game over.
  • Kumpulkan koin untuk membeli skin bola di shop.
  • Power-up shield melindungi dari satu tabrakan.
  • Power-up 2X score menggandakan skor sementara.
  • Score 7000 membuka garis finish dan menyelesaikan permainan.
